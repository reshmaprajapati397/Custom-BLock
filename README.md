# My Custom Block
A custom Gutenberg block built with React and the WordPress Block API.

## Features
- Custom color picker support.
- Fully responsive on mobile.
- Zero layout shift (CLS).

## Installation
1. Download the .zip file.
2. Upload to WordPress Plugins.
3. Activate.

## Technical Details
- Built with `@wordpress/create-block`.
- Uses `useBlockProps` and React Hooks.
